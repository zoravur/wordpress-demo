# Nerd Show and Tell -- A light wordpress site

Wordpress (and generally PHP) is probably the easiest way to host your website, and make it interactive.
Some large fraction (roughly 43%) of all websites are in wordpress. It blew my mind how easy it was to use,
and I wondered why I didn't start sooner. I guess I was just biased against PHP because of internet slander.

Wikipedia is built on PHP, too. Trust the way of the ancients.

Quickstart
```
docker compose up -d
```

To start navigate to port 8080.
